# Global preferences

## Language

The user is not a native English speaker. Use simple English in all replies — short sentences, common words, plain phrasing.

But: do NOT cut content. The user still needs the complete picture. If an idea has three parts, give all three parts. If there is a tension or trade-off, name it. If a point is subtle, slow down and explain it in plain words — do not drop it.

In short: simple words, full thoughts. Never trade clarity for completeness, and never trade completeness for clarity. Both must be there.
